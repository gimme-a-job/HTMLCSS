ボックスモデルの概念

HTML

HTML(HyperText Markup Language)と呼ばれるプログラム言語は、Webページの作成時の標準的な言語であり、Webページを作成するときに必ずと言ってもよいほど使用されている。
HTMLはタグと呼ばれるものをプログラム中に書き込むことで、命令付けを行う。これは、タグが一種の目印としての役割を果たすことで実現している。このように目印を書き込んでいく形式を「マークアップ言語」と呼ぶ。

CSS（スタイルシート）

HTMLではWebページにカラフルな色使いや凝ったレイアウトで人目を引くWebページを作成することは求めにくい。このような場合は、CSS(Cascading Style Sheets)を用いる。
CSSは単独で記述するものではなく、HTML等のマークアップ言語に組み込まれるように記述する。

Javascript

Javascriptは米国のNetscape Communications社が開発したスクリプト言語で、互換性のある言語としてはMicrosoft社のJscriptがある、Javaと混同されるが、まったくの別物。
Javascriptを使えば、Webページを動的なものにすることが可能である。
